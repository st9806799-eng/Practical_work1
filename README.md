# Практична робота №1: Основи Web-дизайну

## Про проєкт
У цьому репозиторії представлені матеріали з курсу веб-дизайну. 
Мета роботи: розробка адаптивного інтерфейсу та вивчення основ композиції.

## Використані інструменти
* Figma — прототипування та дизайн макетів.
* Adobe Photoshop — робота з растровою графікою.
* HTML/CSS — верстка базових елементів.

### 1. Дослідження та аналіз (UX)
* Визначення цільової аудиторії проєкту.
* Аналіз конкурентів та пошук референсів на Pinterest/Behance.
* Побудова карти користувацького шляху (User Flow).

### 2. Прототипування
* Створення низькодеталізованих макетів (Wireframes) для визначення структури сторінок.
* Робота над сіткою (Grid) та ієрархією елементів.

### 3. Візуальний дизайн (UI)
* Підбір колірної палітри (акцентні кольори, фони, текстові блоки).
* Вибір типографіки: використання шрифтів [Назви тут свої шрифти, наприклад, Inter або Montserrat].
* Розробка UI-kit: кнопки, форми введення, іконки.

### 4. Адаптивність
* Створення окремих макетів для десктопної, планшетної та мобільної версій.
* Перевірка читабельності контенту на малих екранах.
