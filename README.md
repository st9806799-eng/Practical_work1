# Практична робота №1: Основи Web-дизайну

## Про проєкт
У цьому репозиторії представлені матеріали з курсу веб-дизайну. 
Мета роботи: розробка адаптивного інтерфейсу та вивчення основ композиції.

## Використані інструменти
* Figma — прототипування та дизайн макетів.
* Adobe Photoshop — робота з растровою графікою.
* HTML/CSS — верстка базових елементів.

## Структура репозиторію
* /designs — макети у форматі PNG/JPG.
* /assets — графічні ресурси проєкту.
* index.html — головна сторінка проєкту.
