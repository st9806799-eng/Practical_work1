# Основи Web-дизайну: Теоретичний огляд та Практика

## 📖 Що таке Web-дизайн?
Web-дизайн - це процес проектування візуальної оболонки та логічної структури веб-сайтів. Він поєднує в собі художнє оформлення (естетику) та технічне проектування зручності користування.

---

## 🏗️ Ключові принципи дизайну

### 1. Візуальна ієрархія
Це спосіб розташування елементів таким чином, щоб користувач інтуїтивно розумів, що є найголовнішим. Це досягається через:
* Розмір: Більші елементи привертають увагу першими.
* Колір: Яскраві акценти для кнопок (CTA).
* Контраст: Виділення тексту на фоні.



### 2. Типографіка
Вибір шрифтів безпосередньо впливає на сприйняття бренду та легкість читання. Основні правила:
* Використання не більше 2-3 різних шрифтів.
* Дотримання достатнього міжрядкового інтервалу.
* Чіткий поділ на заголовки (H1, H2, H3) та основний текст.

### 3. Сітки та вирівнювання (Grids)
Сітки допомагають впорядкувати контент, роблячи його структурованим та професійним. Найпопулярнішою є 12-колонкова сітка, яка дозволяє легко адаптувати макет під різні екрани.

## 🎨 Складові UI/UX

| Поняття | Опис |
| :--- | :--- |
| UX (User Experience) | Проектування досвіду: як користувач взаємодіє з сайтом, наскільки це логічно та просто. |
| UI (User Interface) | Проектування інтерфейсу: як сайт виглядає (кольори, кнопки, іконки, шрифти). |

---

## 🛠️ Інструментарій сучасного дизайнера
1. Figma - основний стандарт для створення макетів та прототипів у реальному часі.
2. Adobe Color - сервіс для підбору гармонійних колірних палітр.
3. Google Fonts - бібліотека безкоштовних та якісних шрифтів.
